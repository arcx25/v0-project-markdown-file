Our Mission

ARCHITECT // VAULT is a secure V3 marketplace connecting buyers and vendors anonymously. We provide cryptographically authenticated channels ensuring privacy and security for all marketplace participants without compromising operational security.

{# Updated workflow for buyer/vendor marketplace #}

How It Works

1

Anonymous Registration

Create an account using only a PGP key. No email, no phone, no personal information required. Your cryptographic key is your identity.

2

Post or Browse Opportunities

Buyers post opportunities. Vendors browse listings and express interest. The system facilitates matching without revealing identities until both parties agree.

3

Review and Match

Verified vendors submit proposals. Buyers review credentials and proposals, then choose their preferred vendor to work with.

4

Communicate Securely

All messages are end-to-end encrypted with PGP. Even we cannot read your communications. Transaction details shared through encrypted channels only.

Security Architecture

PGP-Based Authentication (ARCHITECT Protocol)

No passwords. Users authenticate using cryptographic challenge-response with their PGP keys.

End-to-End Encryption

All messages are encrypted with the recipient's public key. The platform cannot read message contents.

Tor Integration

Access via Tor hidden service ensures maximum anonymity and protection against network surveillance.

Anonymous Payments

Monero-based donations provide untraceable financial support for whistleblowers.

How to Use

For Sources (Whistleblowers)

  1. Generate a PGP key pair if you don't have one
  2. Register with your public key
  3. Submit leads describing your information
  4. Review journalist interest and select a match
  5. Communicate securely through encrypted messages

For Journalists

  1. Register with your PGP public key
  2. Complete verification process (organization, credentials)
  3. Browse active leads
  4. Express interest with a pitch
  5. If selected, communicate securely with the source
