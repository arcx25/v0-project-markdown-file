About ARCHITECT // VAULT

Mission

ARCHITECT // VAULT is a secure platform designed to connect whistleblowers with verified journalists through cryptographically authenticated, end-to-end encrypted channels.

Security Architecture

PGP-Based Authentication (ARCHITECT Protocol)

No passwords. Users authenticate using cryptographic challenge-response with their PGP keys.

End-to-End Encryption

All messages are encrypted with the recipient's public key. The platform cannot read message contents.

Tor Integration

Access via Tor hidden service ensures maximum anonymity and protection against network surveillance.

Anonymous Payments

Monero-based donations provide untraceable financial support for whistleblowers.

How to Use

For Sources (Whistleblowers)

  1. Generate a PGP key pair if you don't have one
  2. Register with your public key
  3. Submit leads describing your information
  4. Review journalist interest and select a match
  5. Communicate securely through encrypted messages

For Journalists

  1. Register with your PGP public key
  2. Complete verification process (organization, credentials)
  3. Browse active leads
  4. Express interest with a pitch
  5. If selected, communicate securely with the source

Open Source

ARCHITECT // VAULT is open source software. The codebase is available for audit and contribution.
